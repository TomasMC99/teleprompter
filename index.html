Connect a Bluetooth joystick/controller before opening a script.

Button mapping:
• Button 0 (A / ✕) — Play / Pause
• Button 1 (B / ○) — Scroll faster (+10)
• Button 2 (X / □) — Scroll slower (−10)
• D-pad Up / Left stick up — Scroll backward
• D-pad Down / Left stick down — Scroll forward
• Left axis — Fine speed control

Pair your joystick via iPhone Bluetooth settings first, then connect here.
